熵 = state function。三句心法：(i) ΔS 沿 任何可逆路径 计算，与实际不可逆路径无关；(ii) 储热源 ΔS = ±Q/T_reservoir，温度用储热源的 T；(iii) ΔS_universe ≥ 0，等号仅当全过程可逆。

一页速览

  • P1 三源热机. 可逆 ⟺ ΔS_univ = 0，给最大 W；不可逆给 ΔS_univ > 0。能量守恒 \(W=Q_{\text{abs}}-Q_{\text{rel}}\) + 熵守恒一起解三个未知 \((Q_3,W,\eta)\)。
  • P2 矿弹. 第一阶段：等容 + 全部内能增加 ΔU=Q ⟹ T' ; ΔS_gas = nC_v ln(T'/T)。 第二阶段：等温等外压（不是可逆）膨胀，做功 \(W=P(V'-V)=nR(T'-T)\)；用可逆路径 ΔS_gas = nR ln(V'/V) = R ln(T'/T)。
  • P3 冰 + 水. 平衡温 给定 ⟹ 反解 T_i；熵分三段算（融化 \(mL/T\) + 新水升温 \(mc\ln(T_f/0)\) + 原水降温 \(Mc\ln(T_f/T_i)\)），单位换 J/K (cal→J)。
  • P4 房屋熵. 房屋稳态 → ΔS_house = 0；外界吸 Q 给 ΔS_out = Q/T_out。瞬态降温阶段两者都要算。"下限"参数：友人作复杂热源时，所有外部不确定贡献 ≥ 0，所以 ΔS_univ ≥ ΔS_out。

Problem 1 — 三源热机 Cycle / ΔS_univ

循环系统每循环吸 300 J（来自 300 K 储库）、吸 50 J（来自 100 K 储库）、释放 Q（到 200 K 储库）。
(a) 最大每循环做功 W 是多少？
(b) 作为热机，效率 η？
(c) 若改为吸 20 J（来自 100 K）+ 300 J（来自 300 K），却仍输出 (a) 中的 W。可能吗？用能量和熵论证。

(a) 最大 W：可逆情形

关键：循环 ΔU = 0、ΔS_system = 0（state function 回到初态）。所以宇宙熵全部来自三个储库：

$$\Delta S_{\text{univ}}=-\frac{Q_{300}}{300}-\frac{Q_{100}}{100}+\frac{Q_{200}}{200}.$$

（符号：储库放热进系统时储库失熵，记 -Q/T；储库吸热加熵，记 +Q/T。）

最大 W ⟺ 最少 Q 释放到 200K ⟺ ΔS_univ = 0（可逆极限）：

$$-\frac{300}{300}-\frac{50}{100}+\frac{Q_{200}}{200}=0\quad\Longrightarrow\quad Q_{200}=200\cdot 1.5=300\ \text{J}.$$

能量守恒：W = 入 - 出 = (300+50) - 300 = 50 J。

⚠ 陷阱：用 ΔS_univ ≥ 0，等号给 最大 做功

第二定律要求 ΔS_univ ≥ 0。越接近 0，越接近可逆，越接近最大做功。这是热机优化的基本逻辑。 若你写成"ΔS_univ ≤ 0 给最大功"，那是反了——这种方向给的是"physically impossible"。

⚠ 陷阱（高频）：用 Q-加权算术平均温度套 Carnot 是错的

常见错误想法："把两个热源合成等效温度 \(T_{\text{avg}}=\frac{Q_1 T_1+Q_2 T_2}{Q_1+Q_2}\)，然后用 Carnot \(\eta=1-T_c/T_{\text{avg}}\)。"

代入：

$$T_{\text{avg}}=\frac{300\cdot 300+50\cdot 100}{350}\approx 271.4\ \text{K},\quad\eta=1-\frac{200}{271.4}\approx 26.3\%,$$ $$W_{\text{错}}=350\cdot 0.263\approx 92\ \text{J}.$$

真值是 50 J，偏高 84%！

错因（数学）：Carnot 公式 \(\eta=1-T_c/T_h\) 的来源是熵守恒 \(Q_h/T_h=Q_c/T_c\)，T 在 分母。 多储库合成等效温度，正确做法是对 1/T 做 Q-加权算术平均，等价于 T 的 调和平均：

$$\frac{1}{T_{\text{eff}}}=\frac{\sum_i Q_i/T_i}{\sum_i Q_i}=\frac{300/300+50/100}{350}=\frac{1.5}{350}\quad\Longrightarrow\quad T_{\text{eff}}\approx 233.3\ \text{K}.$$ $$\eta=1-\frac{200}{233.3}=\frac{1}{7},\quad W=\frac{350}{7}=50\ \text{J}.\ \checkmark$$

由 AM-HM 不等式，算术平均 ≥ 调和平均，所以算术法永远高估 η 和 W。 记忆口诀："温度进熵公式是倒数，所以平均要按倒数加权"。

深层物理图像：把循环拆成"引擎 + 热泵"

本题 100 K 储库 低于 "冷"储库 200 K，意味着从 100 K 吸热再排到 200 K 是反方向（冷→热），这是 热泵 行为，需要消耗功。所以本题真正的物理几何：

  • 引擎 A（300 K → 200 K，可逆）：吸 \(Q_1=300\) J 于 300 K，按 Carnot \(q_1=Q_1\cdot 200/300=200\) J 释于 200 K，输出 \(W_A=100\) J
  • 热泵 B（100 K → 200 K，反向 Carnot）：吸 \(Q_2=50\) J 于 100 K，释 \(q_2=Q_2\cdot 200/100=100\) J 于 200 K，输入 \(W_B=q_2-Q_2=50\) J
  • 200 K 储库总接收 \(q_1+q_2=300\) J ✓
  • 净功 \(W=W_A-W_B=100-50=50\) J ✓

用 50 J 功把 50 J 从 100 K"抬"到 200 K，剩下 50 J 是 net engine output。把 300 K 和 100 K 合成"平均热源"等同于把引擎和热泵融合，几何上不成立。

通用算法（≥ 3 储库 / 温度排布不寻常时永远适用）

遇到多储库循环 不要试图套 Carnot 简化版。直接列两个守恒律：

$$\underbrace{\sum_i\frac{-Q_i^{\text{in}}}{T_i}+\frac{Q_{\text{out}}}{T_{\text{out}}}=0}_{\text{熵守恒（可逆）}}\quad+\quad\underbrace{W=\sum_i Q_i^{\text{in}}-Q_{\text{out}}}_{\text{能量守恒}}.$$

两条方程解两个未知数 \((Q_{\text{out}}, W)\)，对任意温度排布、任意方向都对——包括有储库低于"冷"端、有多个并列热源、有等温段嵌套等等。

(b) 效率 η

η = W / Q_absorbed。系统吸热 = 300 + 50 = 350 J（两个储库都给系统供热）。

$$\eta=\frac{50}{350}=\frac{1}{7}\approx 0.143=14.3\%.$$

⚠ 陷阱：分母用 总吸热 不是其中之一

有两个源同时供热，分母应 = 全部吸入的热（即 350 J），不是其中某一个（300 J 或 50 J）。 η = W / Q_abs 的定义中 Q_abs 是 系统吸入的总热。

(c) 改为 20 J + 300 J 时

能量论证

给定 W = 50 J（不变），输入 = 300 + 20 = 320 J，输出 = 输入 - W = 320 - 50 = 270 J 到 200 K 储库。

熵论证

$$\Delta S_{\text{univ}}=-\frac{300}{300}-\frac{20}{100}+\frac{270}{200}=-1-0.2+1.35=+0.15\ \text{J/K}.$$

ΔS_univ > 0 ⟹ 符合第二定律 ⟹ 物理上可能。

结论：这种情形下循环 不可逆（如果可逆 ΔS_univ 应该 = 0）。物理上：减少了 30 J 来自冷源 100 K 的供热，却没补偿能输出更多功，多出的"能量缺口"反映在更多 Q 释放到 200 K 储库上，产生熵增。

"可逆" = "理想效率"

对比 (a) 与 (c)：相同的 W、相同的 300K 输入热，但 (c) 少吸 30 J 来自 100 K，多排 30 J 到 200 K。 ΔS_univ 多了 30·(1/200 - 1/100) = 30·(-1/200) = -0.15 J/K... 等等，让我重算。 实际上：(c) 比 (a) 多排 270-300 = -30 J（少排 30 J） + 少吸 30 J 来自 100 K。 变化量 ΔS 比 (a) 增加：少排到 200K ⟹ -30/200 = -0.15；少吸 100K ⟹ +30/100 = +0.30。 所以 (c) 比 (a) 多 +0.15 J/K，0+0.15=0.15 J/K ✓

(a) W_max = 50 J (Q_200 = 300 J 释放) · (b) η = 1/7 ≈ 14.3% · (c) 物理上可能 (ΔS_univ = +0.15 J/K > 0)；循环 不可逆

Problem 2 — 矿弹爆炸 + 等外压膨胀 Bomb + Isothermal

1 mol 空气在汽缸内，活塞，初平衡 T = 273 K（内外压相等）。
(a) 隔热绝缘 + 活塞夹死。微型炸弹爆炸释放 1000 J 热。新温 T' = ?
(b) (a) 中气体熵变 ΔS (J/K)？
(c) 接下来汽缸接触温度为 T' 的储库，同时松开夹具。气体抵抗外压膨胀到新平衡。求 ΔS_gas、ΔS_universe (J/K)？

预备：空气 = 双原子，C_v = 5R/2 = 20.775 J/(mol·K)，C_p = 7R/2 = 29.085 J/(mol·K)。

(a) 新温度 T'

等容 + 绝热（除了爆弹这个内热源）：W = 0，ΔU = Q_bomb = 1000 J。

$$\Delta U=n C_v\Delta T\quad\Longrightarrow\quad \Delta T=\frac{1000}{1\cdot(5/2)\cdot 8.31}=\frac{1000}{20.775}\approx 48.13\ \text{K}.$$ $$T'=273+48.13\approx 321.13\ \text{K}.$$

⚠ 陷阱：空气 = 双原子，C_v = 5R/2

Quiz 5 公式表明确给"diatomic c_v = 5/2 R"。空气主要是 N₂、O₂ 都是双原子。 用单原子 3/2 R 会让 T' 跳到 354 K，错。

(b) ΔS_gas (等容升温)

用恒容版熵公式（V₂ = V₁ 时第二项消失）：

$$\Delta S=n C_v\ln\frac{T'}{T}=1\cdot\frac{5}{2}\cdot 8.31\cdot\ln\frac{321.13}{273}=20.775\cdot\ln 1.176=20.775\cdot 0.1626\approx 3.38\ \text{J/K}.$$

为什么 ΔS_gas > 0？

气体温度升高 → 微观态数量 Ω 增加（动量空间球壳半径 ∝ √U 变大），所以 S = k ln Ω 上升。 虽然 Q 来自"内部"（炸弹），但热能注入仍是熵的来源，等于 Q = nC_v ΔT 在 T 域上的 ∫dQ_rev/T = nC_v ln(T'/T)。 这是 "internal Q" 还是 "external Q" 不影响 ΔS_gas——因为熵是 state function，初终态决定。

(c) 等外压膨胀

初态（c 段开始时）：T = T'，V = V₁，P_int = nRT'/V₁。 末态：T = T'（接触 T' 储库），活塞自由，P_int = P_ext = P_atm = nRT/V₁（注意 P_atm = 初始 a 段前的内压，= nRT/V₁ 即 273 K 时的内压）。

新体积

P_atm = nRT/V₁ = nRT'/V₁'，所以

$$V_1'=V_1\cdot\frac{T'}{T}=V_1\cdot\frac{321.13}{273}\approx 1.176\,V_1.$$

ΔS_gas — 用 state function 沿可逆等温路径

初终态都在 T = T'，所以 ΔU = 0，膨胀 V₁ → V₁'：

$$\Delta S_{\text{gas}}=nR\ln\frac{V_1'}{V_1}=R\ln\frac{T'}{T}=8.31\cdot 0.1626\approx 1.35\ \text{J/K}.$$

⚠ 陷阱：实际过程不可逆，但 ΔS_gas 计算用可逆路径

实际：活塞松开瞬间 P_int = nRT'/V₁ > P_atm，气体快速冲出对外做功，非准静态。 但 ΔS 是 state function ⟹ 与路径无关 ⟹ 用任何 可逆 路径计算 ΔS 都给出同样的答案。 最简单的可逆路径：等温（T' 储库）准静态膨胀。沿它 dS = δQ_rev/T = (PdV)/T = nR dV/V，积分给上面那个公式。

Q 实际值 & ΔS_reservoir

实际过程（非可逆等外压膨胀）：

  • ΔU = 0（等温）
  • W_gas = P_ext · (V₁' - V₁) = P_atm · V₁(T'/T - 1) = nRT(T'/T - 1) = nR(T' - T) = 8.31·48.13 ≈ 400 J
  • Q（从储库给气体）= ΔU + W = 0 + 400 = 400 J

储库失热 400 J，温度 T' = 321 K（可视为大储库不变）：

$$\Delta S_{\text{reservoir}}=-\frac{Q}{T'}=-\frac{400}{321.13}\approx -1.246\ \text{J/K}.$$

宇宙：

$$\Delta S_{\text{univ}}=\Delta S_{\text{gas}}+\Delta S_{\text{reservoir}}=1.35-1.25\approx 0.10\ \text{J/K}.$$

ΔS_univ > 0 ⟹ 这是不可逆过程 ✓（验证：活塞瞬间释放，气体非准静态冲出，确实不可逆）。

熵会计三栏表

实体实际能量熵变

Problem 3 — 冰 + 水 熵 Phase change

1 kg 冰 (0°C) 投入装 10 L (= 10 kg) 水的桶。平衡后温度 = 20°C。
(a) 水的初始温度 (°C)？
(b) 宇宙熵变 (J/K)？

(a) 反解水的初温

能量守恒（用 kcal）：

  • 冰融化：吸 \(Q_1 = m_i L_f = 1\cdot 80 = 80\) kcal
  • 新水从 0°C 升到 20°C：吸 \(Q_2 = 1\cdot 1\cdot 20 = 20\) kcal
  • 原水从 T_i 降到 20°C：放 \(Q_3 = 10\cdot 1\cdot(T_i-20)\) kcal

守恒：Q₁ + Q₂ = Q₃ → 100 = 10(T_i - 20) → \(T_i = 30°\text{C}\)。

(b) 宇宙熵变

三步全部用绝对温度 K，比热和潜热换成 J/K 单位。\(1\ \text{cal}=4.184\) J，\(c_w=4184\) J/(kg·K)，\(L_f=80\cdot 4184=334720\) J/kg。

第 1 步：冰融化 (T = 273 K 恒定)

$$\Delta S_1=\frac{m_i L_f}{T_{\text{melt}}}=\frac{1\cdot 334720}{273}\approx 1226\ \text{J/K}.$$

第 2 步：新水从 273 K 升到 293 K

沿可逆加热路径（与外部 0→20°C 一系列 T 储库可逆接触）：

$$\Delta S_2=m c_w\ln\frac{T_f}{T_i}=1\cdot 4184\cdot\ln\frac{293}{273}=4184\cdot 0.07076\approx 296\ \text{J/K}.$$

第 3 步：原水从 303 K 降到 293 K

$$\Delta S_3=M c_w\ln\frac{T_f}{T_i}=10\cdot 4184\cdot\ln\frac{293}{303}=41840\cdot(-0.03356)\approx -1404\ \text{J/K}.$$

合计

$$\Delta S_{\text{univ}}=\Delta S_1+\Delta S_2+\Delta S_3=1226+296-1404\approx 118\ \text{J/K}.$$

正值 ✓（自发的热混合过程必定 ΔS_univ > 0）。

⚠ 陷阱：T 必须用 Kelvin 单位

S = ∫dQ/T 中的 T 是 绝对温度，必须用 K 不能用 °C。 若用 °C：0°C = 0 → \(\Delta S_1 = mL_f/0\) → 无穷大（错）。 "ΔT 用 °C/K 数值相等"只适用于 \(\kappa A\Delta T/L\) 这类 ΔT 公式，不适用于 1/T 的积分。

⚠ 陷阱：可逆路径不等于"实际接触一个储库"

实际过程：1 kg 冰直接接触 30°C 水，瞬间没有"温度梯度无穷小的可逆接触"。 但 ΔS 是 state function，初终态决定。计算时假装"沿可逆路径"用 ln(T_f/T_i) 公式即可。 若题目改问"具体过程的 Q/T 积分"，答案不同——因为该积分 ≤ ΔS。但本题问 ΔS 直接，用公式。

验证：单位与数量级

融化吸热 ≈ 334720 J，主导熵 ΔS₁。原水冷却释放熵主导 ΔS₃。两个绝对值都 ~1300 J/K，差出来才是宇宙增熵 ≈ 116 J/K（约 10% 量级，合理）。

(a) T_i = 30°C · (b) ΔS_univ ≈ 118 J/K (or ~116 J/K with rounding)

Problem 4 — 房屋宇宙熵 House ΔS

Q4 P4 的房屋问题延续。加热器电源 3333 J/s 维持房间 25°C（外 -25°C）。
(a) 1 小时宇宙熵变？
(b) 加热器坏掉，21 分钟内温度跌到 20°C。期间宇宙熵变？(近似流出 ≈ 3333 J/s 不变)
(c) 31 朋友到达，发热 3200 J/s，室温稳定 23°C。每小时宇宙熵变下限？
提示：熵是 state function。

关键温度：T_out = -25°C = 248 K（恒定）。室内为 25°C = 298 K (a)；20→25°C 之间 (b)；23°C = 296 K (c)。

(a) 稳态加热 1 小时

稳态：T_house 不变 → ΔS_house = 0（state function，初终态相同）。 电力来自插座：电能 → 热能 → 流出。电网不发热产 entropy 这部分不需要算（也无法算）。 外界吸收 Q = 3333·3600 = 1.2×10⁷ J 在 T_out = 248 K：

$$\Delta S_{\text{out}}=\frac{Q}{T_{\text{out}}}=\frac{1.2\times 10^7}{248}\approx 48387\ \text{J/K}\approx 4.84\times 10^4\ \text{J/K}.$$ $$\Delta S_{\text{univ}}=\Delta S_{\text{house}}+\Delta S_{\text{out}}\approx 4.84\times 10^4\ \text{J/K}.$$

⚠ 陷阱：稳态 ⟹ ΔS_house = 0，不要算成 nC_v·something

稳态意味着 初终态完全相同（同 T、同 V、同 P、同分子分布）。熵作为 state function 自然不变。 电热器把 1.2×10⁷ J 的电能（高品质能量）转为 1.2×10⁷ J 的低温热能（低品质能量），这就是宇宙增熵的来源。

(b) 加热器坏掉 21 分钟，T 从 25°C 跌到 20°C

近似 Q̇ 不变 = 3333 J/s（室内 T 变化小）：

21 分钟 = 1260 s。Q 流出 = 3333·1260 = 4.20×10⁶ J 到外储库 (248 K)。

$$\Delta S_{\text{out}}=\frac{4.20\times 10^6}{248}\approx 16935\ \text{J/K}.$$

室内：从 298 K 降到 293 K。空气作为系统，n = PV/RT = 40918 mol（用初态 T = 298 K），C_v = 5/2·R：

但要用 nC_v 还是 Q/ΔT？两个略不同（前者 850000 J/K，后者 840000 J/K）。题目暗示用"3333 J/s 的近似 + 5 K 跌幅"，所以用 Q/ΔT：

$$C=\frac{Q}{\Delta T}=\frac{4.20\times 10^6}{5}=8.4\times 10^5\ \text{J/K}.$$ $$\Delta S_{\text{house}}=C\ln\frac{T_f}{T_i}=8.4\times 10^5\cdot\ln\frac{293}{298}=8.4\times 10^5\cdot(-0.01693)\approx -14216\ \text{J/K}.$$ $$\Delta S_{\text{univ}}=\Delta S_{\text{house}}+\Delta S_{\text{out}}\approx -14216+16935\approx 2719\ \text{J/K}.$$

正值（不可逆热传导）。

对比 (a) 与 (b)

(a) 稳态运行 1 小时 ΔS_univ ≈ 48,387 J/K——大量电能持续转热外排，整体熵增很大。 (b) 21 分钟瞬态 ΔS_univ ≈ 2,719 J/K——室内损失自身的 14216 J/K（变冷），但外界获得 16935 J/K，差额是不可逆部分。 若房屋能"完全可逆"地把内部 5K 温降转为热到外界，理论上 ΔS_univ 应该 = 0；但热传导本质不可逆 → 永远有差额。

(c) 朋友们 + 稳态 23°C 一小时下限

稳态 23°C：

  • Q̇_out = κA·ΔT/L = 0.1·200·(23-(-25))/0.3 = 0.1·200·48/0.3 = 3200 J/s
  • 友人 + 自己发热 = 3200 J/s ✓ 平衡

1 小时输出热量 Q = 3200·3600 = 1.152×10⁷ J 到外储库。

$$\Delta S_{\text{out}}=\frac{1.152\times 10^7}{248}\approx 46452\ \text{J/K}.$$

室内 + 友人 + 食物 + 代谢链 = 复杂系统。但稳态意味着 室内气体熵不变，ΔS_house = 0。 友人 + 食物 + 代谢的总熵变 ≥ 0（任何"内部"不可逆过程都增熵）：

$$\Delta S_{\text{univ}}=\underbrace{\Delta S_{\text{house}}}_{=0}+\Delta S_{\text{out}}+\underbrace{\Delta S_{\text{ppl+metabolism}}}_{\ge 0}\ge \Delta S_{\text{out}}.$$ $$\boxed{\Delta S_{\text{univ}}\ge 4.65\times 10^4\ \text{J/K}.}$$

⚠ 陷阱："下限"语义

题目要 lower bound，意思是 ΔS_univ 不可能 低于 这个值。 理论上 ΔS_ppl 可能很大（实际代谢有大量熵增）但 至少 ≥ 0。所以保守下限 = 仅 ΔS_out 这部分（其余视为 ≥ 0）。 若题目问"上限"或"精确值"，需要更多生物 / 食物链信息。

对比 (a) 与 (c)

(a) 电热源稳态 1 小时：48387 J/K；(c) 朋友热源稳态 1 小时：≥ 46452 J/K。两者非常接近因为流出功率仅差 4%（3333 vs 3200）。 区别：(a) 的电源转化是单一不可逆步骤（电→热）；(c) 涉及食物 → 化学能 → 代谢 → 热能 的多步代谢链，每一步都增熵——所以 (c) 真实 ΔS_univ 应该 显著 大于这个下限。

陷阱汇总

本卷 9 大易错点

  1. T 在 dS=δQ/T 必须用 Kelvin；用 °C 会爆炸（除零）。
  2. 循环系统 ΔS_system = 0（state function 回到初态），熵增全在储库。
  3. "可逆路径"是为了计算 ΔS 的数学工具；实际过程不可逆不影响计算。
  4. 多源热机：ΔS_univ ≥ 0，等号给最大功；不等号给现实不可逆运行。
  5. 多储库循环不能套 2 储库 Carnot 简化版！错用 Q-加权算术 T_avg 会高估 W（本题偏 84%）。正确：直接列 ΔS_univ=0 + 能量守恒。
  6. η = W / Q_abs，分母用总吸入热（多个源都算）。
  7. 空气是双原子，c_v = 5R/2 不是 3R/2。
  8. 稳态系统 ΔS = 0：电源稳定加热的房间熵不变（初终态相同）。
  9. "下限"问题：把不确定的内部贡献视为 ≥ 0，给保守估计。
